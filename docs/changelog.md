# v 3.0.3 (2018-08-03)

- 增加组件：Select - 下拉框
- 增加组件：NoticeBar - 通告栏
- 增加组件：Steps - 步骤条
- 增加组件：Calendar - 日历
- 增加组件：Accordion - 手风琴
- 增加组件：Result - 结果页
- 更新组件：Rater - 评分 新增属性允许半选、清除、触摸移动选中等功能，具体使用方法请参考对应文档
- 更新组件：Qrcode - 二维码 修复相对路径的问题
- 更新组件：Toptips - 顶部提示 函数调用后，返回值支持 promise 接口
- 更新组件：Toast - 提示框 函数调用后，返回值支持 promise 接口
- 更新组件：Notification - 通知 函数调用后，返回值支持 promise 接口
